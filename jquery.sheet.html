Did you know?
  • jQuery.sheet supports multi spreadsheets
  • Within a single sheet, you can reference cell values by using the syntax like "A1" or a group of cells like this "A1:B4"
  • Within multi sheets, you can reference cell values from other sheets by using the syntax like "SHEET2:A1" or a group of cells like this "SHEET2:A1:B4"
  • jQuery.sheet is licensed under Dual licensed under the LGPL and GPL licenses (http://www.gnu.org/licenses)
  • jQuery.sheet uses jQuery UI Themeroller, which lets you change your theme
  • jQuery.sheet has built in methods to export the sheet as
    • HTML: jQuery.sheet.export.html()
    • XML: jQuery.sheet.export.xml()
    • jSON: jQuery.sheet.export.json()
  • When coding with jQuery.sheet, you can use it's shorthand "jS", example: jS.openSheet(tableObjects)
  • jQuery.sheet won't effect html around it
  • jQuery.sheet and it's spreadsheet cell location bars are resizable (dependant on jQuery UI)
